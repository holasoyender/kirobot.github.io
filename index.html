deKiro

Bot de discord customizable y en Español

Como configurarme

1 - Obtén el Archivo de configuración con el comando exportar

2 - Configura tu archivo usando un editor como Visual Studio Code o similares.

3 - Comprueba tu archivo usando un Validador de YAML

4 - Importa tu archivo usando el comando importar y listo!

Información

deKiro es un bot creado por holasoyender#8934 y MicroondasV2#4508 y hosteado por holasoyender#8934

Entre sus módulos destacan:

• Módulo de moderación: con todos los elementos esenciales para la moderación.

• Módulo de información: con muchos comandos de información útiles como el comando k/recordar.

• Módulo de ocio: con una gran cantidad de comandos muy variados para entretener a los usuarios con diferentes minijuegos.

• Módulo de música: para reproducir todo tipo de videos de YouTube en el chat de voz.

• Módulo de sorteos: para sortear un premio en tu servidor.

• Módulo de logs: para registrar todos los eventos en tu servidor.

• AutoMod: Este modulo lo puedes personalizar en el archivo de configuración para que borre todas las palabras que quieras y todos los enlaces de invitación.
