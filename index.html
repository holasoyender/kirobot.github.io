deKiro

Bot de discord customizable y en Español

Como configurarme

1 - Obtén el Archivo de configuración con el comando exportar

2 - Configura tu archivo usando un editor como Visual Studio Code o similares.

3 - Comprueba tu archivo usando un Validador de YAML

4 - Importa tu archivo usando el comando importar y listo!

Módulo de música

deKiro cuenta con un módulo de música único para escuchar tus canciones favoritas sin parar y con la mejor calidad de sonido.

Moderación avanzada

Olvidate de los spammers gracias al módulo de moderación automática de deKiro y despreocupate!

Sorteos y autoroles

Con deKiro puedes alojar todo tipo de sorteos en tu servidor y hacer que los usuarios escojan sus roles con el menú de autoroles con botones!
